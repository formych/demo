# go-http

聚合一些go版本的http demo

## 模块

### gin

一个基于`gin`的简单`http`服务

用于验证docker, kubernetes, istio的各种功能的http demo

### 使用proto进行数据序列化

`proto`目录演示基于`http`, 使用`protobuf`进行序列化数据

### 使用多种方式启动http服务

`listen`目录演示基于`http`包启动http服务

### 直接基于tcp

`tcp`实现直接用tcp启动服务，读取数据

### 数据交流

`http`演示进行自定义读取数据

### 表单数据

`form`的数据提取和默认解析
